== Description ==
A simple Anti Spam for Comment by number or including character.


== Installation ==

e.g.

1. Upload `hjyl-comment-spam` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You can choose if you need the Anti Spam include character by plugin admin page.
